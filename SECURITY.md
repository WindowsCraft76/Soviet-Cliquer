**Last updated:** 2026-06-28

| Version    | Supported          |
| ---------- | ------------------ |
| 0.3.0      | :white_check_mark: |
| < 0.3.0    | :warning:          |

:white_check_mark: **Supported** - :warning: **Not supported** (Safe) - :x: **Not supported** (Not safe)

## Reporting a Vulnerability

If you find a security vulnerability in a supported version of the application, please report it [here](https://windowscraft76.fr/sovietcliquer/r/issues/)!

## Version Numbering System

The versioning system consists of three numbers in the following format:

`Release.Beta.Hotfix`

Example: `1.2.5`

### Meaning of the numbers

- **First number** - Release

Represents a major version of the application.
Example: `1.0.0` or `2.0.0`

- **Second number** - Beta

Represents an update for a new development phase of the current version. The additions/changes in these versions are not final.
Examples: `0.1.0` or `1.1.0`

- **Third number** - Hotfix

Represents a quick fix for bugs or minor issues from the previous version.
Examples: `1.0.1` or `0.1.1`
